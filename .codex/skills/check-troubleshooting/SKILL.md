# 트러블슈팅 검색

## 목표

처음부터 다시 진단하기 전에 관련 애플리케이션 트러블슈팅 문서를 찾는다.

## 작업 순서

1. `docs(haru2)/troubleshooting/`를 검색한다
2. 아래 기준으로 매칭한다:
   - 키워드
   - 증상
   - 발생 조건
3. 상위 1~3개 문서를 짧은 이유와 함께 반환한다

## 규칙

- 문구 일치보다 증상 일치를 우선한다
- 관련 문서가 없으면 명확히 없다고 말한다
- 관련 문서를 문서에 기록하거나 링크할 때는 작성 중인 문서 위치 기준 상대경로로 쓴다
- 로컬 절대경로 링크는 쓰지 않는다
- 이 단계에서는 새 문서를 만들지 않는다
