# App Reference

실행 절차는 아니지만 루트 애플리케이션 작업에서 자주 참조하는 비교표, 규칙, 배경 설명, 상위 workflow 기준을 이 디렉터리에 둔다.

이 디렉터리의 문서는 `document-writing-rules.md`를 기준으로 유지한다.

## 문서 목록

- `application-code-change-summary.md`
- `application-dev-vs-prod.md`
- `application-source-of-truth.md`
- `application-workflow-from-idea-to-operation.md`
- `document-writing-rules.md`
- `operational-notes.md`
- `repo-workflow-parity.md`

## 사용 원칙

- 절차가 되기 시작하면 `../runbooks/`로 보낸다
- 장애 재현/해결 기록이면 `../troubleshooting/`로 보낸다
- 현재 작업 상태와 진행 기록은 `../status/`, `../tasks/`로 보낸다
- 루트 애플리케이션의 source of truth, workflow 기준, 상위 작업 메모는 이 디렉터리에 둔다
- 인프라 운영 절차나 운영 상태 스냅샷은 `infra/` 아래로 보낸다
