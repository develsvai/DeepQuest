## 의도

- `infra/docs`를 기준본으로 삼아 루트 `docs(haru2)`와 `.codex/skills/`의 골격을 애플리케이션 작업 레이어에 맞게 다시 정렬한다

## 현재 상태

- 루트 `AGENTS.md`와 `.codex/skills/`는 `docs(haru2)`를 canonical docs 경로로 보도록 다시 맞춰졌다
- 공통 workflow skill은 `infra` 쪽의 더 성숙한 규칙을 앱 레이어에 맞게 옮겨 왔고, 부트스트랩, 상대경로 링크, task 작성 스타일 규칙이 반영됐다
- `infra/`는 최신 기준본이며, 루트는 `infra`의 문서 골격과 기능을 참고하되 인프라 전용 개념은 직접 복제하지 않는 방향으로 정리하고 있다

## 알려진 이슈

- 루트 최신 상태 문서와 작업 문서가 오래되어 현재 작업 축을 충분히 반영하지 못했다
- 루트 `docs(haru2)/reference/`에는 `infra/docs/reference/`에 대응하는 상위 workflow 기준, source of truth, 운영 메모 축이 비어 있었다
- 기존 루트 문서 일부에는 `docs/` 기준 경로 표기가 남아 있다
