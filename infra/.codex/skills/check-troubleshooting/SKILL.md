# 트러블슈팅 검색

## 목표

처음부터 다시 진단하기 전에 관련 인프라 트러블슈팅 문서를 찾는다.

## 작업 순서

1. `docs/troubleshooting/`를 검색한다
2. 아래 기준으로 매칭한다:
   - 키워드
   - 증상
   - 발생 조건
3. 상위 1~3개 문서를 짧은 이유와 함께 반환한다

## 규칙

- 문구 일치보다 증상 일치를 우선한다
- 관련 문서가 없으면 명확히 없다고 말한다
